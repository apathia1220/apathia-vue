Navbar #

层级选项卡 | 多级标签页

TIP

Navbar 是为了处理有层级的 tab。若是简单的一级 tab 建议用 Tabs 组件。

Navbar 暂仅支持两层，不支持三层。

基础用法 #

<template>
  <p>nav: {{ nav }}</p>

  <Navbar v-model="nav" :list="list" :trigger="'click'" />
</template>

<script>
import { ref, defineComponent, reactive } from 'vue'

export default defineComponent({
  setup() {
    const nav = ref(1)
    const list = reactive([
      {
        label: 'nav1',
        value: 1,
      },
      {
        label: 'nav2',
        value: 2,
      },
      {
        label: 'nav3',
        value: 3,
        children: [
          {
            label: '二级1',
            value: '3-1',
          },
          {
            label: '二级2',
            value: '3-2',
          },
          {
            label: '二级3',
            value: '3-3',
          },
        ],
      },
    ])
    return {
      nav,
      list,
    }
  },
})
</script>

子级触发方式 #

通过指定 trigger 属性为 click 指定子级必须点击才生效，默认为 hover

<template>
  <p>current nav: {{ nav }}</p>

  <Navbar v-model="nav" :list="list" trigger="click" />
</template>

<script lang="ts">
import { defineComponent, ref, reactive } from 'vue'

type NavNode = {
  value: string | number
  label: string
  disabled?: boolean
  iconClass?: string
  children?: NavNode[]
}

export default defineComponent({
  setup() {
    const nav = ref<number>(1)
    const list = reactive<Array<NavNode>>([
      {
        label: 'nav1',
        value: 1,
      },
      {
        label: 'nav2',
        value: 2,
      },
      {
        label: 'nav3',
        value: 3,
        children: [
          {
            label: '二级1',
            value: '3-1',
          },
          {
            label: '二级2',
            value: '3-2',
          },
          {
            label: '二级3',
            value: '3-3',
          },
        ],
      },
    ])

    return {
      list,
      nav
    }
  }
})
</script>

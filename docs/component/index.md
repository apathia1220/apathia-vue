# Component

## 单独安装

::: tip 全局安装
可以单独安装某一个组件使用
:::

```bash
# 比如安装 `Modal` 组件
yarn add @apathia/apathia.modal
```

```js
import { useModal, Modal } from '@apathia/apathia.modal'
```

## 全局导出

::: tip 导出
也可以直接从 `@apathia/apathia` 中直接导出
:::

```js
import { useModal, Modal } from '@apathia/apathia'
```

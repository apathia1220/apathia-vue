# Hooks

## 常用的 composition api

::: tip 单独安装
可以单独安装使用 `@apathia/apathia.hooks`包
:::

```bash
yarn add @apathia/apathia.hooks
```

```js
import { useEventListener } from '@apathia/apathia.hook'
```

## 全局导出

::: tip 导出
也可以直接从 `@apathia/apathia` 中直接导出
:::

```js
import { useEventListener } from '@apathia/apathia'
```
